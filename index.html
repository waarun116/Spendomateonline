‚ùì Frequently Asked Questions

How do I set up my PIN and what are the security requirements?

When you first open the app, you'll be prompted to set up a 4-digit PIN. The app includes advanced security validation to ensure your PIN is secure:

  • No repeated numbers (e.g., 1111)
  • No sequential numbers (e.g., 1234)
  • Minimum length of 4 digits

You'll also have the option to set up a security question for PIN recovery.

What happens if I forget my PIN? How does the recovery process work?

If you forget your PIN, you can use your security question to reset it. Here's how the process works:

  • Tap "Forgot PIN?" on the login screen
  • Answer your security question correctly
  • Set up a new secure PIN

Note: If you haven't set up a security question, you'll need to reset the app, which will clear all local data.

How is my data protected and what security measures are in place?

Your data is protected with multiple layers of security:

  • SHA-256 encryption for all stored data
  • Local storage only - no data is sent to external servers
  • Secure PIN validation and security question authentication
  • Brute force protection with automatic lockout
Can I use biometric authentication and what devices are supported?

Currently the app security runs on security pin authentication:

  • Minimum 4 digit security pin non-sequential
  • Security question system both custom and default of user choice

The app will soon upgraded to allow biometric authentication for better approach.

What happens if I enter the wrong PIN or security answer too many times?

To protect your data from unauthorized access, the app includes a security lockout feature:

  • After 5 incorrect attempts, the app will be locked
  • The lockout period is 1 hour
  • This applies to both PIN and security question attempts

This feature helps prevent brute force attacks on your data.

How do I set up and manage my security question?

You'll be prompted to set up a security question when you first create your PIN. Here are some tips:

  • Choose a question that only you know the answer to
  • Make sure the answer is something you can remember
  • Answers are case-sensitive
  • You can't change the question after setup

Remember: Your security question is crucial for PIN recovery.
